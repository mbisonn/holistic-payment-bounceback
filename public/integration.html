Tenera Integration Successfully Loaded

The Tenera Holistic Payment Hub Integration is now active on this page. You can use the global teneraPayment object to interact with the integration.

Available Methods

  • teneraPayment.checkout() - Process cart and redirect to checkout
  • teneraPayment.getCartData() - Get the current cart data
  • teneraPayment.clearCart() - Clear all cart data
  • teneraPayment.setCheckoutUrl(url) - Update the checkout URL

Integration Snippet

Add this script to your sales page:

<script src="https://www.teneraholisticandwellness.com/landingPageIntegration.js" type="module"></script>

For older browsers that don't support ES modules:

<script src="https://www.teneraholisticandwellness.com/landingPageIntegration.legacy.js"></script>
